# Estrutura do projeto

Este projeto e uma loja estatica para GitHub Pages, com HTML, CSS e JavaScript gerados a partir do script:

`scripts/build-impacto360-premium-mall.mjs`

Arquivos principais:

- `pacote-github-pages-pronto/index.html`: pagina principal para hospedagem.
- `pacote-github-pages-pronto/404.html`: copia da loja para rotas do GitHub Pages.
- `pacote-github-pages-pronto/dados/products.json`: produtos e servicos exibidos na loja.
- `pacote-github-pages-pronto/dados/importedMercadoLivreProducts.json`: links importados do Mercado Livre.
- `pacote-github-pages-pronto/dados/mercado-livre-links.json`: links originais extraidos do arquivo enviado.
- `pacote-github-pages-pronto/dados/stores.json`: lojas internas do shopping.
- `pacote-github-pages-pronto/public`: imagens da fachada e placeholders.

Para publicar no GitHub Pages, envie o conteudo da pasta `pacote-github-pages-pronto` para a raiz do repositorio.
