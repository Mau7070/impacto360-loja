# ADENDO — NOVAS DIRETRIZES PARA INCLUIR NO COMANDO CODEX

Inclua as seguintes diretrizes no comando principal do robô enriquecedor do Shopping Impacto 360:

1. Ajustar a loja para abrir perfeitamente em telemóvel/celular, com design mobile-first.
2. Garantir que a imagem de cada produto apareça inteira, sem corte, usando `object-fit: contain`.
3. Criar botão público grande e visível: “Entrar no Shopping Impacto 360”.
4. Criar painel administrativo protegido com botão “Abrir loja ao público”, visível somente para o administrador e oculto ao público.
5. A loja deve começar fechada até que o administrador insira e revise os produtos.
6. Permitir inserir/alterar WhatsApp da loja no painel administrativo.
7. Exibir botão WhatsApp nos produtos somente quando o número estiver configurado.
8. Criar robô de oferta inteligente que cruza a busca do cliente com produtos e contatos autorizados.
9. Enviar no máximo uma oferta por dia para cada cliente.
10. Enviar ofertas somente para contatos com consentimento registrado e respeitar opt-out/SAIR.
11. Preservar sempre o `link_original_preservado` como link de compra afiliado.
12. Gerar fila de ofertas para revisão quando a API do WhatsApp não estiver configurada.
13. Gerar relatórios de envio, bloqueios, contatos sem consentimento e conversões.
