# Atualização do comando — versão 2

Este pacote contém a versão atualizada do comando para o Codex com as novas diretrizes solicitadas:

- loja adaptada para telemóvel/celular;
- imagem integral dos produtos;
- botão público para entrar no Shopping Impacto 360;
- painel administrativo para abrir/fechar lojas;
- WhatsApp configurável;
- robô de oferta inteligente por busca do cliente;
- limite de uma oferta por dia por cliente;
- proteção contra envio sem consentimento;
- preservação do link original de afiliado.

Use o arquivo principal consolidado:

```txt
COMANDO_CODEX_IMPACTO360_COMPLETO_V3.md
```

Se quiser usar somente a parte de loja, painel e robô, use:

```txt
COMANDO_CODEX_ROBO_ENRIQUECEDOR_V2.md
```

Se quiser usar somente a automação de inicialização no Windows, use:

```txt
COMANDO_CODEX_AUTO_START_WINDOWS.md
```

Se quiser apenas complementar o comando antigo, use:

```txt
ADENDO_NOVAS_DIRETRIZES_CODEX.md
```
