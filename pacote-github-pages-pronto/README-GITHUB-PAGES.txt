IMPACTO 360 - PACOTE PRONTO PARA GITHUB PAGES

Este pacote deve ser enviado para a raiz do repositório:

https://github.com/Mau7070/impacto360-loja

Arquivos importantes:

- index.html precisa ficar na raiz do repositório.
- 404.html ajuda o GitHub Pages a não quebrar navegação.
- .nojekyll evita bloqueio de arquivos estáticos.
- public/ contém imagens e logo.
- robots.txt e sitemap.xml já apontam para:
  https://mau7070.github.io/impacto360-loja/

Como publicar:

1. Abra o repositório no GitHub.
2. Clique em Add file > Upload files.
3. Arraste todos os arquivos de dentro desta pasta, não a pasta inteira.
4. Confirme em Commit changes.
5. Vá em Settings > Pages.
6. Source: Deploy from a branch.
7. Branch: main.
8. Folder: /root.
9. Salve.

Link de teste esperado:

https://mau7070.github.io/impacto360-loja/

Se você usa domínio próprio, mantenha o arquivo CNAME existente no repositório e configure DNS no provedor do domínio.
